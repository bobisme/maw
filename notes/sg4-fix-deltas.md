## What this doc captures

Per cluster (one row per T4.1 backlog entry):

- **Baseline cost** — `total_cost_turns` from the pre-hardening
  FrictionList.
- **After cost** — `total_cost_turns` from the post-hardening
  FrictionList.
- **Δ %** — `(baseline - after) / baseline * 100`, two decimals.
  Undefined (`n/a`) when `baseline == 0`.
- **Target** — the per-cluster pass/fail policy: `≥ N% reduction`,
  `reaches 0` (small-baseline rows), or `stays 0`
  (vocabulary-scarcity / new-cluster cells).
- **Verdict** — `MET` / `MISSED` / `REGRESSED`.

Per-cluster decisions are independent — **no composite "fraction of
clusters that met target" headline** (T4.1 hard rule, lifted from
T2.4 / T2.8). The iteration policy fires per-cluster: a missed target
re-opens that specific child bone, not T4.2 wholesale.

The **unattributed bucket delta** is surfaced at top level (NOT as a
row). If its growth exceeds 20%, the T4.1 footnote blocks the SG4
stop-condition independent of per-cluster reductions.

## Iteration triggers (re-open T4.2 children)

_(populated by `just sg4-rebench`. Empty when every cluster met
target and no regressions surfaced.)_

The iteration policy is **per-cluster** (T4.1 hard rule). Each
missed-target or regressed row names a specific fix-task child bone
the lead re-opens; T4.2 itself is NOT re-opened wholesale.

## Renegotiated targets (template; populate only when a target is sound but missed)

When a target is missed and the measurement is sound, document the
renegotiation here. Required fields per renegotiation entry:

- **Cluster + fix-task bone:** e.g. `ws_destroy_refused` / `bn-c6l3`.
- **Original target:** e.g. `reaches 0`.
- **Measured delta:** e.g. `40% reduction (1 → 0.6)`.
- **Renegotiated target:** e.g. `≥ 40% reduction at the next bench`.
- **Rationale:** what structural blocker prevents closing the
  remaining gap.
- **Tracking bone:** ID of the new bone capturing the structural
  follow-up.

**No renegotiation in pilot** — pilot uses synthetic data; the
section exists as a TEMPLATE only.

## Regenerating this file

```bash
# Pilot (planted demo pair, no campaign cost):
just sg4-rebench-pilot

# Real run: takes a baseline + after FrictionList pair, OR two sweep
# artifact directories of BenchRun JSONs (auto-reduced on the fly):
just sg4-rebench notes/baseline-friction-list.json notes/after-friction-list.json

# Direct binary invocation:
cargo run -p maw-bench-metrics --features bench --bin sg4-rebench -- \
  --baseline <path-or-dir> --after <path-or-dir> \
  --out-json /path/to/sg4-fix-deltas.json \
  --out-md   /path/to/sg4-fix-deltas.md
```
